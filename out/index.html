passport.js

the aim of this file is to implement 2 passport strategies that will be used to authenticate requests to the overall API endpoints. When the user logs in our app, this will function as a validation method for the username and password and compare them to our users collection in the database area. The JWT strategy is also used, this will decode the Web Token returned to the user after having a successfull login, after it will check the user ID, and if it matches the user collection in the database.
