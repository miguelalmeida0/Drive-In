Movie API

For this project i will be building a modern movie app, similar to netflix. The users will be able to access all the specific information about the movies provided!

To achieve this task i will be using:

  • HTML
  • CSS
  • Node.js
  • Javascript
  • MongoDB

Essential Features

Request URL HTTP Method Response
Get a list of movies '/movies' GET Returns a JSON list of movies.
Get data about the movie '/movies/:title' GET Returns a JSON data about a specific movie.
Get data about the director '/director/:directorName' GET Returns a JSON data about the author's name.
Get all users '/users' GET Returns a JSON data with all users.
Return data about a genre by name/title. '/genres/:genre' GET Returns a JSON data with movies in that genre.
Allow users to update their user information '/users/:Username' PUT Returns the updated data fields by the user.
Add a new movie to the list of favorites '/favorite/:movieName' POST Returns a text confirming which movie was added successfully.
Allow new users to register '/users/:newUser' POST Will add a new user to a JSON object.
Allow existing users to deregister '/users/:deleteAccount' DELETE Return a text confirming that the user was deleted.
Remove movie from list of favorites '/favorite/:deleteMovie' DELETE Returns a text confirming which movie was removed successfully.
